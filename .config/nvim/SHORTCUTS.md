# Keyboard Shortcuts

## LSP & Diagnostics

| Shortcut | Action |
|----------|--------|
| `<leader>cl` | Check LSP health (`:checkhealth lsp`) |
| `<leader>xx` | All workspace diagnostics (Trouble panel) |
| `<leader>xX` | Current buffer diagnostics only (Trouble panel) |
| `[d` | Jump to previous diagnostic in buffer |
| `]d` | Jump to next diagnostic in buffer |

## Gradle

| Shortcut | Action |
|----------|--------|
| `<leader>Gs` | Gradle Sync |
| `<leader>Gf` | Gradle Sync (Force, ignore cache) |
| `<leader>Gc` | Clear Gradle Cache |

## Android

| Command | Action |
|---------|--------|
| `:AndroidInfo` | Show Android development configuration status |

## General

| Shortcut | Action |
|----------|--------|
| `ggVG` | Select all content in buffer |
